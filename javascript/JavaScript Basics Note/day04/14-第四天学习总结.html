1.1-第四天所学知识点总结

  • 1.熟悉数组的三要素

    • 元素：数组中存储的数据
    • 下标：元素在数组中的位置
      • 数组的下标从0开始，连续存储
    • 长度：数组中元素的数量
      • 最大下标 = 长度 - 1

  • 2.掌握数组的取值与赋值

    • 取值数组[下标]
      • 获取第一个元素：数组[0]
      • 获取最后一个元素:数组[数组.length -1]
      • 如果下标超过数组最大下标，得到的是undefined
    • 赋值：数组[下标] = 赋值
      • 如果下标有对应的值，会把原来的值覆盖
      • 如果下标不存在，会给数组新增元素
      • 给数组最后面添加一个元素:数组[数组.length] = 值
      • 删除数组最后面一个元素:数组.length--

  • 3.掌握数组的长度特点

    • 获取数组的长度：数组.length
    • 设置数组的长度会修改数组的元素

  • 4.熟悉数组的遍历方式

    • 固定的for循环语法：for(var i = 0;i<数组的长度;i++){}

  • 5.掌握数组的冒泡法排序:固定了for循环嵌套

    • 核心思想：数组中相邻的两个元素比较大小，交换位置
    • 1.外层循环决定比较的轮数 ：for(var i = 0;i<数组的长度-1;i++)
      • 为什么数组的长度-1,因为最后一轮无需比例(例如一个数组有五个数字，第五大的数字自然就是最小的哪一个)
    • 2.内层循环决定每一轮比较的次数:for(var j = 0;j<数组的长度-1-i;i++)
    • 3.交换两个相邻元素的位置
      • 如果是从小到大排序:if(数组[j] > 数组[j+1])
      • 如果是从大到小排序:if(数组[j] < 数组[j+1])

  • 6.了解数组的去重：开关法

  • 7.了解数组的另一种创建方式

    • 简洁方式: [元素1.元素2，元素3……]
    • 标准方式： new Array(元素1，元素2，元素3……)
    • 两者唯一的区别：当数组中只有一个元素时含义不同
      • [10]代表数组长度为1，只有一个元素10
      • new Array(10):代表数组长度为10的空数组

  • 8.了解二维数组的使用

    • 数组中每一个元素都是一个数组
    • 取值与赋值：二维数组[行][列]

  • 9.掌握引用类型与值类型的区别：复制粘贴文件(值类型)与创建文件快捷方式（引用类型）的区别

    • 值类型：五种基本数据类型，数据保存在栈中
      • 特点：赋值时拷贝的数据，修改新的数据不会对之前的数据有影响
    • 引用类型:两种复杂数据类型，数据保存在堆中，变量在栈中保存的是堆地址
      • 特点：赋值时拷贝的地址，修改新的数据会对之前的数据有影响

1.3-今日课后作业

  • 1.输入班级人数(比如10人),然后依次输入学员成绩，计算班级学员的平均成绩和总成绩

  • 2.本金10000元存入银行，年利率是千分之三，每过1年，将本金和利息相加作为新的本金。计算5年后，获得的本金是多少？

  • 3.将字符串数组用|或其他符号分割成字符串

    • 例如：['中国','我','爱','你'] 变成：中国|我|爱|你

  • 4.要求将数组中的为0的那些值去掉，将不为0的值存入一个新的数组，生成新的数组

    • 例如:[10,20,30,0,40] 变成[10,20,30]

  • 5.练习：（1）排序对数组进行从大到小排序 （2）翻转数组
